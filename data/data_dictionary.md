# Data Dictionary

This dataset contains publicly available loan-level data used to analyze customer churn in a short-term lending context.

## Key Fields

- customer_id: Unique identifier for a borrower
- loan_id: Unique identifier for a loan
- issue_date: Date when the loan was issued
- loan_end_date: Date when the loan was fully paid or closed
- loan_status: Current status of the loan (paid, defaulted, etc.)
- loan_amount: Principal amount of the loan

Dates and identifiers are derived or normalized from the original dataset where necessary.
